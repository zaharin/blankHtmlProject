# blankHtmlProject

## Quick start

Clone the git repo:

```bash
git clone -o base https://github.com/zaharin/blankHtmlProject.git MyApp
cd .\MyApp
npm install -g gulp
npm install
```

## Features
* less
* jade
* html5
* Typescript
* jQuery v1.10.2
* Modernizr v2.6.2
